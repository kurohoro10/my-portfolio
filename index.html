Web Dev That Doesn’t Break When You Blink

I design, I code, I debug... so you don’t have to.

Building a website is easy. Building one that users understand, trust, and enjoy using? That’s where most businesses get stuck. I’m AJ, a web developer and UI/UX designer who helps brands solve that exact problem. I combine clean, scalable code with user-focused design to create websites that not only look good — but actually work the way your users expect them to. Whether it’s a clunky interface, a confusing user flow, or a site that’s just not converting, I bring a problem-solving mindset to every project. I don’t just write code or push pixels — I build web experiences that feel intuitive, fast, and friction-free. If your site isn’t doing what it’s supposed to do, let’s fix that — from the inside out.
