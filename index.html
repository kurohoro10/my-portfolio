Event Calendar

An interactive event calendar built with FullCalendar.js, jQuery, and Moment.js, featuring localStorage-based event persistence. Users can create, edit, and delete events with details like title, location, and description. The interface includes a pop-up event form, real-time date/time formatting, and an autocomplete search to quickly locate events. Data is saved in the browser’s storage, ensuring events remain available after page reloads without a backend.
